this is fully server side project, handled with laravel blade

# featurs
1. signin , signup and signout
2. admin & user dashboard
3. update user status
4. delete user


# To run this project
Follow these steps to set up and run the project locally:

1. Clone the repository 

    ```bash
    git clone https://github.com/ChamsEldinne/simple-_laravel_app.git
    cd simple-_laravel_app

    ```

2.  Install dependencies & Run database migrations 
    ```bash
    composer install
    php artisan migrate:fresh --seed
    ```
