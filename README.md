# To run this project
Follow these steps to set up and run the project locally:

1. **Clone the repository **

    ```bash
    git clone <repository-url>
    cd <project-folder>

    ```

2. ** Install dependencies & Run database migrations **
    ```bash
    composer install
    php artisan migrate:fresh --seed
    ```
